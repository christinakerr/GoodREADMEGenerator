# Good README Generator

## Walkthrough Video

[Access the walkthrough video here.](http://www.youtube.com/watch?v=m30yAlFbw7E)

## Purpose

This generator builds a professional README to speed up the process for developers. It prompts the user for project title, description, installation instructions, etc. in the command line, then generates a README.md file complete with a license badge and a table of contents.

## Tech

This application leverages:

* JavaScript
* Node.js
* Inquirer.js
